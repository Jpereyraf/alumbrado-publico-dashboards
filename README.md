# Dashboards de Alumbrado Público

Dashboards interactivos desarrollados para el seguimiento y análisis del alumbrado público en la zona de concesión de **EDESUR Dominicana, S.A.**

## Contenido

| Archivo | Descripción |
|---|---|
| [`Dashboard_Lamparas_Sur.html`](./Dashboard_Lamparas_Sur.html) | Dashboard de lámparas para la región Sur. |
| [`Dashboard_Mapa_Lamparas.html`](./Dashboard_Mapa_Lamparas.html) | Mapa interactivo de lámparas de alumbrado público. |

## Cómo verlos

### Opción 1 — Descargar y abrir local
1. Haz clic en el archivo `.html` que quieras ver.
2. En la página del archivo, haz clic en el botón **"Raw"** o descarga el archivo.
3. Ábrelo con cualquier navegador moderno (Chrome, Edge, Firefox).

### Opción 2 — GitHub Pages (recomendado)
Si se activa GitHub Pages en este repositorio, los dashboards se pueden ver directamente en el navegador a través de una URL pública sin necesidad de descargarlos.

Para activarlo:
1. Ve a **Settings → Pages** del repositorio.
2. En **Source**, selecciona la rama `main` y carpeta `/ (root)`.
3. Guarda los cambios.
4. Espera ~1 minuto y los dashboards estarán disponibles en:
   - `https://jpereyraf.github.io/alumbrado-publico-dashboards/Dashboard_Lamparas_Sur.html`
   - `https://jpereyraf.github.io/alumbrado-publico-dashboards/Dashboard_Mapa_Lamparas.html`
